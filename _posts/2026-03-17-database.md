## 오늘 배운 내용 : 데이터베이스와 파이썬을 연동했다.
- Windows 11에서 파이썬을 설치했다.
- python 으로 처음 연동해 보았다. 
- 작품은 아래와 같다.

```python

import mysql.connector

conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="--",
    database="python_db"
)

cursor = conn.cursor()

cursor.execute("update users set name ='태양' where id=10")
conn.commit()

cursor.execute("SELECT * FROM users")
print(cursor.fetchall())

cursor.close()
conn.close()

```
