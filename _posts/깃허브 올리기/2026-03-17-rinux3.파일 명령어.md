# 파일 / 디렉토리 명령어

## 파일 디렉토리

### `ls` (list)
 숨긴파일을 제외한 모든 파일을 보여주며 뒤에 명령어가 나올수가있다.
- ls -a (all): 숨긴파일들까지 다 보여준다.
- ls -l (long): 파일의 권한,소유자,크기 등을 상세하게 보여준다.

### `cd` (change directory)
디렉초리 이동할때 쓴다.

- cd [디렉토리 이름]
- cd .. : 현 위치에서 한 단계 상위폴더로 이동
- cd - : 바로 직전 폴더로 이동
- cd ~ : 사용자의 홈 디렉토리로 이동

### `pwd` (print working directory)
현재위치를 물어보는 명령어이다.

### `cp` (copy)
복사 명령어이다.

- cp -r(recursive) : 폴더 안의 하위 내용까지 복사 
- cp -v(verbose) : 파일이 복사되는 과정(어디로 가는지)를 보여준다.
- cp -i(interactive) : 파일이 이미 있으면 덮어쓰는지 물어본다.
**cp 명령어는 별도의 경고없이 덮어쓰기 때문에 cp -i가 필요하ㅓ다**

### `mv` (move)
mv 는 옮기기 및 이름바꾸기가 가능하다.

mv [원본] [대상]
- `mv file.txt /home/user/documents` -> 파일을 폴더로 이동
- `mv old_name.txt new_name.txt` -> 이름바꾸기

### `rm` (remove)
파일이나 디렉토리 삭제하는 명령어이다.

- rm -r(recursive) : 하위내용까지 삭제
- rm -v(varbose) : 삭제되고 있는걸 알려줌
- rm -i(interactive) : 삭제할건지 다시 한번 물어본다.
- rm -f(force) : 파일 강제삭제

### `mkdir` (make Directoty)
파일을 만들때 사용하는 명령어이다.

- mkdir -p (parents) : 하위(중간) 디렉토리를 만들어준다.
- mkdir -v (verbose) : 디렉토리가 생성될때마다 그 과정을 메시지로 출력한다.
- mkdir -m (mode) : 파일을 만들면서 동시에 권한부여한다.

### `touch`
빈 파일을 생성하거나 파일의 시간을 변경하는 명령어이다.

### `cat` (concatenate)
1. **파일의 내용을 즉시 출력한다.**
- cat [파일이름] -> [파일이름]의 내용을 볼수있다
2. 여러 파일의 내용을 하나로 합친다.
- cat file1 file2 > combined_file  
3. 파일만들기
- cat > new file + ctrl D

> cat은 파일은 mkdir은 디렉토리를 만든다.
{: .prompt-info }

### `head` `tail`
head은 앞쪽 정보만 tail은 뒤쪽 정보만 불러온다

head or tail -n 10

>tail -f(follow) 는 파일 끝을 띄어둔 채로 실시간으로 추가되는 내용을 보여준다.
